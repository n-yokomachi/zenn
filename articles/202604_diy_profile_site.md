# はじめに
プレーリーカード風のプロフィール＆リンク集サイトと、NFCカードを自作します。

プレーリーカードはプロフィールやSNSのリンクなどを登録し、スマホなどから読み取れるNFCカードを作るサービスです。
https://prairie.cards/
私もいつだったかのイベントで作ったものが１枚ありますが、カードデザインとか更新したいなーと思っていたのもあり、この際プロフィールサイト自体も自分で作ってしまおうということで、今回の記事の内容となります。

ちなみにプレーリーカードは安い物で3,980円〜になりますが、自作をするとNFCカード（200円）、シール印刷（200円）の合計400円で一枚作れます。
（実際は最低購入枚数があると思います。AmazonではNFCカード3枚セット（600円）がありました）


作成したサイトは以下
https://vivibio.pages.dev/

またリポジトリはこちら
https://github.com/n-yokomachi/vivibio


# つくりかた

1. Claude Designでサイトデザインを作成
2. Claude Codeで1テーマごとにSuperpowersのサイクルを回して実装
3. NFCカードを作る
4. カード完成！　そして・・・

## 1. Claude Designでサイトデザインを作成

まずサイトのデザインはClaude Designを使用しました。
Claude Designについては以下で説明しています。
https://zenn.dev/yokomachi/articles/202604_claude_design_chottodake

初めはいくつか案を出してもらって、そのうちの一つを採用しようと思っていたのですが、自分が優柔不断＆気分で好みが変わるので、せっかくだからいろんなテーマを切り替えられるようにすることとしました。
また、デザインに関してもシンプルなものは一つだけとし、他はアクセシビリティ無視でアニメーションゴリゴリ派手派手を目指して作成させています。
![](https://static.zenn.studio/user-upload/457b1ab8e20e-20260429.png)


## 2. Claude Codeで1テーマごとにSuperpowersのサイクルを回して実装

デザインのベースができたところで実装はClaude Codeに任せます。
エージェントフレームワークにはSuperpowersを使用します。

https://github.com/obra/superpowers
Superpowersはコーディングエージェント向けの開発方法論をAgent Skillsとして配布するプラグインです。
Superpowersを使用した開発では、まず`/brainstorming`スキルによってエージェントがユーザーに対してヒアリングし、相互にインタラクトしながら要求・要件をspecとして整理します。
その後specをもとにplanを作成し、タスクを細分化します。
実装フェーズではサブエージェントを使用して開発とAIレビューを繰り返す実装、またはインラインで開発しチェックポイントごとに人間がレビューする実装の2通りが選択できます。

タスクがかなり細分化されるのとそのタスクごとに2種類のレビューやテスト駆動開発が行われるため、実装そのものの速度は遅いSuperpowersですが、`/brainstorming`がエージェントとのペアプロ感を感じたり、実際アウトプットの品質が良かったり、あとspecやplanファイルが最低限しか作られないのが気に入っています。

というわけで今回は、作ったテーマを1つずつ`/brainstorming`からの実装サイクルにかけて段階的に増やしていきました。

作成したWebサイトはCloudflare Pagesでホスティングしています。


## 3. NFCカードを作る

プレーリーカード的な体験を目指しているのでオリジナルNFCカードを作ります。
オリジナルデザインのNFCカードを作るサービスなどは多分色々あると思いますが、今回は手っ取り早く自分でシールを作って無地のNFCカードに貼って作ります。

というわけでベースのNFCタグ埋め込みカードはAmazonで購入します。
https://www.amazon.co.jp/dp/B0CGMFY5D7?ref=ppx_yo2ov_dt_b_fed_asin_title

プロフィールサイトのURLをスマホアプリでカードのNFCに書き込めば、機能としてはもう完成です。


カードのデザインについてはClaude Codeでいい感じの案を出してもらって作ります。

:::message
カードデザインもClaude Designを使えれば良かったんですが、Webサイトデザインで週間リミットに達してしまいました。Claude DesignのトークンやリミットはClaude Codeとは別物っぽいですが、その分めちゃめちゃトークン食います。今回Claude Designにデザインのベースだけ作らせて、細かい調整はClaude Code側で行ったのもトークン節約目的です。
:::

いくつか案を出してもらった中から、せっかくなので裏表で違うデザインにすることにしました。
![](https://static.zenn.studio/user-upload/5377478d8777-20260429.png)

作った画像はコンビニプリンターでシール紙に印刷し、NFCカードに貼り付けます。
![](https://static.zenn.studio/user-upload/d4ca5b174b1a-20260429.jpeg)

## 4. カード完成！　そして・・・

無事カードができました！
![](https://static.zenn.studio/user-upload/42e6352f74b0-20260429.jpeg)
ぱっと見はちゃんと発注して作ったものみたいな感じにはできたんじゃないかと思います。
これでいつでもどこでも、このカードを差し出せばプロフィールページに飛んでもらうことができます。


・・・ん？　いつでもどこでも？
いやそもそもカードを差し出すということ自体がなんか手間ではないか？
名刺交換と同じ工程になってしまっていないか？

というわけでもっと本質的に「いつでもどこでも」を実現するため、常に身につけている腕時計（Xiaomi Smart Band）にもNFCタグを貼り付けます。
![](https://static.zenn.studio/user-upload/91830cae35a1-20260429.jpeg)

これで本当に、いつでもどこでも手を差し出すだけで自己紹介できるようになりました。
